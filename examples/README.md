## Usage

```bash
# Copy example configuration
cp examples/configuration/cursor-ide-example.json config/cursor-ide.json

# Run example workflow
./examples/workflows/pdf-conversion-demo.sh

# Test example script
./examples/scripts/monitoring-example.sh
```
